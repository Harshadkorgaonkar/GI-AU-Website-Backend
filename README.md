# GI-AU Registration System

## 📌 Project Overview

GI-AU Registration System is a web-based application developed to manage registration data related to GI-tagged products in the Bodoland region. The system helps maintain information about local producers, artisans, and products involved in the GI registration process.

The application provides features for managing user records, product details, reports, and registration data, making the overall registration process more organized and efficient.

---

## 🚀 Features

* User Authentication
* Dashboard Management
* Product Management
* Member Management
* Staff Management
* Report Generation
* Profile Management
* Search Functionality
* Responsive User Interface

## ⚙ Installation


```bash

```

### Backend

```bash

```
